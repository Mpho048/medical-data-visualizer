# Medical Data Visualizer

This project is part of the **Data Analysis with Python** certification from freeCodeCamp. It involves exploring and visualizing medical examination data using **Pandas**, **Matplotlib**, and **Seaborn**.

## Project Objectives
- **Data Cleaning**: Filter out incorrect data (such as blood pressure readings where diastolic is higher than systolic).
- **Feature Engineering**: Add an `overweight` column by calculating the Body Mass Index (BMI).
- **Data Normalization**: Standardize cholesterol and glucose data to binary values (0 for good, 1 for bad).
- **Statistical Visualization**:
    - Create a **Categorical Plot** to visualize health metrics.
    - Create a **Heat Map** to show the correlation between different medical variables.

## Technologies Used
- **Python 3**
- **Pandas** (Data manipulation)
- **Seaborn** (Advanced statistical plotting)
- **Matplotlib** (Base plotting library)

## Dataset
The dataset contains 70,000 records of patients, including features like age, height, weight, blood pressure, and lifestyle habits (smoking, alcohol intake, physical activity).
